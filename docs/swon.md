# SWON – Subject-Weights-Orchestration-Nullification

Аббревиатура, связывающая четыре столпа архитектуры:

- **S**ubject — субъектность, смена ролей;
- **W**eights — выработка весов субъектности;
- **O**rchestration — роевая оркестровка;
- **N**ullification — обнулёнка.

SWON-модель позволяет описывать любые коллективные системы, где участники осознанно меняют свои роли.
