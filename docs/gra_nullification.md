# GRA-Обнулёнка в SubjectSwap

## Принцип

Не ресет данных, а **ценностно-ориентированное обнуление политик и весов** с последующим восстановлением в сторону большей совокупной ценности/любви.

## Процесс

1. Определить индекс ценности $L_i$ для каждого правила, роли, веса.
2. Если средняя ценность роя падает ниже порога — запустить обнуление.
3. Удалить текущие политики, роли, веса.
4. Сгенерировать новые, максимизируя $\sum L_i$.
5. Продолжить эволюцию.

## Код

```python
def nullify(swarm):
    old_value = swarm.evaluate_love()
    swarm.reset_policies()
    swarm.reset_weights()
    swarm.optimize_for_max_love()
    new_value = swarm.evaluate_love()
    assert new_value >= old_value
```
