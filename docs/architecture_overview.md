# Архитектурный обзор

```
┌─────────┐     ┌──────────────────────┐     ┌───────────────┐
│ Optimus │     │ SubjectSwapEngine    │     │ HumanPolicy   │
│  Env    │◄───►│                      │◄───►│ (инструмент)  │
└─────────┘     │  - step()            │     └───────────────┘
                │  - swap()            │
                │  - weights           │
                └──────────┬───────────┘
                           │
                    ┌──────▼──────┐
                    │ GRANullifier│
                    └─────────────┘
```

Компоненты слабо связаны: можно заменить политику человека на GUI, модель робота на RL‑агента, GRA‑нуллификатор на кастомную логику.
