# Архитектура системы

## Компоненты

- **Агент (`Agent`)** — ИИ-сущность, способная быть субъектом или инструментом.
- **Человек (`Human`)** — живой участник, подчиняющийся правилам политики.
- **Политика (`Policy`)** — формальный контракт: субъектности, роли, ограничения.
- **Рой (`Swarm`)** — оркестр агентов и людей, шагающий во времени.
- **Веса субъектности (`Weights`)** — вектор, вычисляемый и оптимизируемый.
- **Обнулитель (`Nullifier`)** — модуль, реализующий GRA-Обнулёнку.

## Шаг роя

1. Применяется текущая политика $\Pi_t$.
2. Определяются субъект и инструмент.
3. Инструмент (человек или агент) выполняет предписанные действия.
4. Происходит диалог, записывается история.
5. Вычисляются новые веса $W_{t+1}$.
6. Если условие обнуления выполнено — вызывается Nullifier.
7. Политика обновляется: $\Pi_{t+1} = \text{SwapRoles}(\Pi_t)$.

## API (псевдокод)

```python
swarm = Swarm([agent1, human])
swarm.step()  # один такт смены ролей
weights = swarm.get_weights()
swarm.nullify_policies()  # принудительная обнулёнка
```
