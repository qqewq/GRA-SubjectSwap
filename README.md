# GRA-SubjectSwap

Фреймворк для динамического свапа субъектности между человеком и агентом (роботом, ИИ).  
Реализует цикл **«Optimus ↔ тренеры»**, где робот учится быть субъектом, а человек – инструментом, и наоборот.

## Установка
```bash
pip install -e .
```

## Быстрый старт
```bash
python examples/optimus/optimus_minimal.py
```
